# Setup

## Pre-requisites 

1. Install [vagrant](https://www.vagrantup.com/)
2. Install [VirtualBox](https://www.virtualbox.org/)
3. Optional for offline
  - download `workshop.box`
  - `vagrant box add royragsdale/avengercon-taoccc workshop.box`

## Start demo box

```
vagrant up
```
